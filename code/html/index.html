Most frameworks try to do everything.
CameronJS does just enough.

CameronJS takes plain HTML, CSS and Javascript and adds just enough smart bits to make building websites as easy as it should be. When you're ready, you can access and create your own APIs with serverless functions via Netlify.

CameronJS is a simple framework for awesome apps.
